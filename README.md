# 1v1 Tetris Rounded Board

🎮 **1v1 테트리스 게임**  
HTML5 Canvas 기반으로 만들어진 브라우저용 테트리스 게임입니다.  
두 플레이어가 동시에 플레이 가능하며, 각자의 보드가 둥근 테두리로 표시됩니다.

---

## 🎯 특징
- 2인용 테트리스 대전
- 각 플레이어의 점수와 최고점 표시
- 블록 하드드랍 위치를 **반투명 고스트 블록**으로 미리 보여줌
- 점수를 올리면 블록이 잠깐 흔들리는 **Shake Effect**
- 게임 오버 시 승리자 표시 및 재시작 가능

---

## ⌨️ 컨트롤

| 플레이어 | 이동 | 회전 | 소프트드랍 | 하드드랍 |
|----------|------|------|------------|-----------|
| P1       | W/A/S/D | W | S | Shift |
| P2       | 방향키 | ↑ | ↓ | / |

- 게임 오버 후 **R 키**를 눌러 재시작 가능
